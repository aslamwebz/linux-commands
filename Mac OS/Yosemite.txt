Variables de Entorno
--------------------
source ~/.bash_login

Servicios
---------
Listar: brew services list


Virtualenvwrapper
-----------------
Instalar: sudo pip install virtualenvwrapper

Variables:
nano ~/.bash_login
export WORKON_HOME=~/Envs
source /usr/local/bin/virtualenvwrapper.sh


PostgreSQL
----------
Instalar: brew install postgresql
Iniciar: brew services start postgresql
Crear db (no necesita contraseña): createdb db
Entrar (no necesita contraseña): psql db


Vagrant
-------
Aprovisionar: vagrant provision
SSH: vagrant ssh
Iniciar: vagrant up


Docker
------
Instalar:
  brew cask install dockertoolbox
  docker-machine create --driver virtualbox docker

Habilitar docker:
  docker-machine create --driver virtualbox docker
  eval "$(docker-machine env docker)"
