BASICOS
*******
- Editar contraseña:
  $ passwd
  
- Editar contraseña usuario:
  $ passwd <<nombre>>
  
- Agregar usuario:
  # useradd <<nombre>>
  
- Remover usuario:
  # userdel <<-r>> <<nombre>>
  
- Crear grupo:
  # groupadd <<nombre>>
  
- Eliminar grupo:
  # groupdel <<nombre>>
  
- Agregar usuario a grupo:
  # gpasswd -a <<usuario>> <<grupo>>
  
- Remover usuario de grupo:
  # userdel <<usuario>> <<grupo>>
  
- Problemas de dependencia:
  # apt-get -f install

- Actualizar repositorios:
  # apt-get update

- Limpiar repositorios:
  # apt-get clean
  
- Actualizar sistema:
  # apt-get upgrade
                 
- Codificar archivo a utf8:
  $ iconv -f windows-1252 -t utf-8 <<filename.from>> > <<filename.to>>
  
- Renombrar dispositivos (ext4):
  # e2label <<dispositivo>> <<etiqueta>>
  
- Lista de dispositivos montados:
  $ mount -l

- Buscar:
  $ find <<ruta>> -name <<nombre>> -print 2>/dev/null 
  
- Probar puerto:
  # telnet <<ip>> <<puerto>>
  
- Ver servicios:
  # service --status-all
  
- Agregar servicio:
  # update-rc.d <<servicio>> defaults
  
- Remover servicio:
  # update-rc.d -f <<servicio>> remove
  
- Permisos de un servicio:
  # chmod 655 /etc/init.d/<<servicio>>
  
- Listar conexiones de red:
  $ netstat -lnt
  
- Zona Horaria:	
  # dpkg-reconfigure tzdata
  
- Seguimiento de archivo:
  $ tailf <<ruta archivo>>
  
- Acceder como sudo:
  $ sudo su -
  
- Abrir como root aplicaciones graficas:
  $ kdesudo <<comando>>
       
- Empaquetar y comprimir tar:
  $ tar czvf archivo.tar.gz /archivo/mayo/*
  
- Desempaquetar y descomprimir tar:
  $ tar xzvf archivo.tar.gz
  
- Ver el contenido (sin extraer) tar:
  $ tar tzvf archivo.tar.gz

- Md5 de archivo:
  $ md5sum <<archivo>>
 
- Instalar empaquetado
  # dpkg -i <<paquete.deb>>
  
- Listar discos
  # fdisk -l
  
- Escribir iso a usb
  # dd if=<<ruta iso>> of=/dev/sd<<letra del usb>>
  
- Especificaciones del cpu
  $ lscpu

POST INSTALAR
*************
* Configurar sudo
  - Agregar linea a "/etc/sudoers": %sudo   ALL=(ALL:ALL) ALL
  # gpasswd -a <<usuario>> sudo
  
* Activar auto completar del terminal
  - Descomentar el "bash completion" en "/etc/bash.bashrc"
  
* Montar partición ntfs con el sistema
  # mkdir /mnt/archivos
  # blkid
  - Agregar a /etc/fstab: UUID=F4CE3C4DCE3C0A7E /mnt/archivos ntfs-3g defaults 0 0
  # mount -a

* Configurar repositorios
  # cp /etc/apt/sources.list /etc/apt/sources.list.original
  - Comentar cdrom
  - Agregar lineas:
  deb http://ftp.us.debian.org/debian/ jessie main contrib non-free
  deb-src http://ftp.us.debian.org/debian/ jessie main contrib non-free
