BASICOS
*******
- Editar contraseña:
  $ passwd
  
- Editar contraseña usuario:
  $ passwd <<nombre>>
  
- Agregar usuario:
  # useradd <<nombre>>
  
- Remover usuario:
  # userdel <<-r>> <<nombre>>
  
- Crear grupo:
  # groupadd <<nombre>>
  
- Eliminar grupo:
  # groupdel <<nombre>>
  
- Agregar usuario a grupo:
  # gpasswd -a <<usuario>> <<grupo>>
  
- Remover usuario de grupo:
  # userdel <<usuario>> <<grupo>>
  
- Problemas de dependencia:
  # apt-get -f install

- Actualizar repositorios:
  # apt-get update

- Limpiar repositorios:
  # apt-get clean
  
- Actualizar sistema:
  # apt-get upgrade
                 
- Codificar archivo a utf8:
  $ iconv -f windows-1252 -t utf-8 <<filename.from>> > <<filename.to>>
  
- Renombrar dispositivos (ext4):
  # e2label <<dispositivo>> <<etiqueta>>
  
- Lista de dispositivos montados:
  $ mount -l

- Buscar:
  $ find <<ruta>> -name <<nombre>> -print 2>/dev/null 
  
- Probar puerto:
  # telnet <<ip>> <<puerto>>
  
- Ver servicios:
  # service --status-all
  
- Agregar servicio:
  # update-rc.d <<servicio>> defaults
  
- Remover servicio:
  # update-rc.d -f <<servicio>> remove
  
- Permisos de un servicio:
  # chmod 655 /etc/init.d/<<servicio>>
  
- Listar conexiones de red:
  $ netstat -lnt
  
- Zona Horaria:	
  # dpkg-reconfigure tzdata
  
- Seguimiento de archivo:
  $ tailf <<ruta archivo>>
  
- Acceder como sudo:
  $ sudo su -
  
- Abrir como root aplicaciones graficas:
  $ kdesudo <<comando>>
       
- Empaquetar y comprimir tar:
  $ tar czvf archivo.tar.gz /archivo/mayo/*
  
- Desempaquetar y descomprimir tar:
  $ tar xzvf archivo.tar.gz
  
- Ver el contenido (sin extraer) tar:
  $ tar tzvf archivo.tar.gz

- Md5 de archivo:
  $ md5sum <<archivo>>
 
- Instalar empaquetado
  # dpkg -i <<paquete.deb>>
  
- Listar discos
  # fdisk -l
  
- Escribir iso a usb
  # dd if=<<ruta iso>> of=/dev/sd<<letra del usb>>
  
- Especificaciones del cpu
  $ lscpu

POST INSTALAR
*************
1. Configurar sudo
    - Agregar linea a "/etc/sudoers": 
      %sudo   ALL=(ALL:ALL) ALL
    - gpasswd -a <<usuario>> sudo
2. Configurar repositorios
    - cp /etc/apt/sources.list /etc/apt/sources.list.original
    * Manera A
    - Agregar lineas a "/etc/apt/sources.list":
      deb http://ftp.debian.org/debian/ wheezy main contrib non-free
      deb-src http://ftp.debian.org/debian/ wheezy main contrib non-free      
    - Comentar cdrom
    - Guardar
    * Manera B    
    - apt-get install netselect-apt
    - netselect-apt -n wheezy
    - mv sources.list /etc/apt/sources.list
3. Activar auto completar
    - Descomentar el "bash completion" en "/etc/bash.bashrc"
4*. Activar multiarch si es 64 bit
    - dpkg --add-architecture i386
5. Actualizar repositorios
    - apt-get update
6*. Intalar linux headers
